# group-chat-app.
Welcome to our group chat application! Our platform combines HTML, CSS, PHP, Ajax, and MySQL to provide a real-time and interactive group chat experience. Let's explore the features of our application: note : application is a simplified demonstration and does not include advanced features like private messaging, file sharing, or user profiles.
